Learning Resources

Additional materials to support your machine learning journey.

Recommended Tools

  • Python: The primary language for ML. Install via Anaconda or Miniconda.
  • Jupyter Notebooks: Interactive development environment for experimentation.
  • Scikit-learn: Essential library for classical ML algorithms.
  • PyTorch / TensorFlow: Deep learning frameworks for neural networks.
  • RDKit: Chemistry toolkit for molecular machine learning.

External Resources

  • Coursera: Andrew Ng's Machine Learning and Deep Learning specializations
  • Fast.ai: Practical deep learning courses with a top-down approach
  • Kaggle: Competitions and datasets for hands-on practice
  • Papers With Code: Latest research papers with implementation code
