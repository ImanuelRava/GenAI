Key Features & Integrations

What makes TMC the most complex section

Chemical Intelligence
SMILES notation support, role classification (reactant, ligand, catalyst, product), color-coded badges, reaction scheme visualization.
SMILES RDKit 3Dmol.js
BYO API Key System
User-provided API keys stored in localStorage. Provider-specific model selection. No server-side key storage for privacy.
localStorage Client-side Privacy-first
Async Support
Both sync and async endpoints for all LLM operations. Flask async routes with HTTP-based LLM client for better performance.
async/await HTTP Client Rate Limiting
File Handling
PDF upload with text extraction, Excel export for citation data, drag-and-drop zones, file preview capabilities.
PDF.js SheetJS Base64
Interactive Visualizations
Cytoscape.js for networks, D3.js for knowledge graphs, 3Dmol.js for molecules. Zoom, pan, click interactions.
Cytoscape D3.js 3Dmol
Performance Features
Rate limiting (20 req/min for chat, 10 req/min for KG), Flask-CORS, response caching, lazy loading, progress indicators.
Flask-Limiter Memory Cache CORS

PDF Processing
PyMuPDF (fitz) for PDF-to-image conversion (DPI=150), pypdf/pdfplumber for text extraction with fallback chain.
