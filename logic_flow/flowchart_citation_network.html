DOI Extraction & Paper Details
2.1
Extract DOI from PDF
DOI module extracts the Digital Object Identifier from the PDF's first page
from modules.DOI import extract_doi_from_pdf main_doi = extract_doi_from_pdf(pdf_path) # Example: "10.1021/jacs.0c12345"
2.2
Fetch Paper Details
Query OpenAlex and Crossref APIs for comprehensive paper metadata
author, year, citations, refs, title = get_paper_details(main_doi) # Returns: author, publication year, citation count, references, title
3
Network Building (3 Types)
Forward Reference
Find papers that cited this paper
1. Query OpenAlex for citing papers
2. Use Crossref for DOI validation
3. Add citing papers as nodes
4. Create edges: citing → main
Backward Reference
Find papers this paper cited
1. Extract references from paper
2. Crossref DOI lookup
3. OpenAlex for metadata
4. Create edges: main → reference
Cross Reference
Find interconnections between papers
1. Load both cited & citing papers
2. Build bidirectional edges
3. Identify highly connected nodes
4. Generate network statistics
3.1
Forward: Build Citing Network
build_forward_network() queries OpenAlex & Crossref
citing_papers = get_forward_citations(main_doi) for paper in citing_papers: G.add_node(doi, author=..., year=..., citations=...) G.add_edge(citing_doi, main_doi) # Arrow points TO main paper
3.2
Backward: Build Reference Network
build_reference_network() uses Crossref for DOI resolution
ref_dois = get_referenced_dois(main_refs) for doi in ref_dois: author, year, cites, _, title = get_paper_details(doi) G.add_node(doi, ...) G.add_edge(main_doi, doi) # Arrow points FROM main paper
