Available Modules

These technical modules provide researchers with powerful tools for literature analysis, knowledge discovery, and data extraction. Each module is designed to accelerate specific aspects of computational chemistry research.

Citation Network

Build interactive citation graphs from a single paper. Analyze forward citations (who cites this paper), backward references (what this paper cites), or cross-reference networks from a list of DOIs.

Knowledge Graph

Generate AI-powered knowledge graphs for any chemistry topic. Enter a keyword or upload a PDF to visualize concepts, reactions, and their relationships in an interactive D3.js graph.

Data Extraction

Extract chemical reactions, compounds, and conditions from scientific PDFs. Supports vision-based extraction (figures + schemes), text-based extraction (ChemExtract AI), and ReactionLens paragraph screening.

Bring Your Own API Key

All AI-powered modules support 8 LLM providers (DeepSeek, OpenAI, Anthropic, Gemini, Groq, HuggingFace, OpenRouter, Ollama). Configure your preferred provider and API key directly in each tool's interface — no server-side keys required.

Citation Network

Navigate the complex web of scientific publications through our interactive citation graph.

The citation network visualization enables researchers to explore the interconnected landscape of chemistry research. Identify seminal works, trace the evolution of ideas, and discover new connections between research areas.

Features

  • Forward analysis — find papers that cite your uploaded paper
  • Backward analysis — find references cited by your uploaded paper
  • Cross-reference — build a network from a list of DOIs in an Excel file
  • Interactive Cytoscape.js visualization with drag and zoom
  • Suggested related papers based on citation impact and local relevance
  • Powered by Crossref and OpenAlex APIs

How It Works

Upload a PDF and choose an analysis type. The tool extracts the DOI, queries Crossref for paper metadata and OpenAlex for forward citations, then builds an interactive network graph with suggested high-impact references.

Data Extraction

Automatically extract chemical reactions, compounds, and conditions from scientific literature PDFs.

We utilize AI-powered tools to automate the extraction of chemical information from scientific literature. The collected data is structured and can be stored in a searchable database.

Capabilities

  • Vision extraction — analyze embedded figures and scheme pages with vision LLMs
  • ChemExtract AI — comprehensive pipeline combining vision + text extraction
  • ReactionLens — text-driven paragraph screening for reaction data
  • Text extraction — raw PDF text extraction with cascading fallback (pypdf → pdfplumber → PyMuPDF)
  • Named entity recognition for chemical compounds
  • Reaction condition extraction (temperature, time, solvents)
  • Yield and product identification
  • R-group table assembly and SMILES formatting

Data Sources

We process publications from major chemistry journals including JACS, Angewandte Chemie, Organic Letters, and more. Results are cached on-disk by PDF content hash for fast repeat analysis.
