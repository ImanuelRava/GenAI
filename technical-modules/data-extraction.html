Extraction Mode

ChemExtract

Fast text + vision extraction pipeline

Extracts reactions, compounds, conditions from PDF text and images. Outputs SMILES reaction schemes (e.g. CC(=O)O.CCO>>CC(=O)OCC). Best for quick extraction from well-structured papers.

New

ReactionLens

Text-driven paragraph screening

Extracts text from the paper, segments into paragraphs, then screens each one for chemical reactions (conditions, reagents, yields, etc.). Outputs the same ChemExtract-compatible format with reactions, compounds, SMILES, R-group tables, and conditions.
